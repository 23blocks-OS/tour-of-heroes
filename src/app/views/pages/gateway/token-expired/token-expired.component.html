Cambio de Contraseña

Si solicitaste un Token para cambiar tu contraseña hace algún tiempo, tu Token ha expirado.

También pudo pasar que solicitaste el cambio de contraseña varias veces. Asegúrate de que estas usando el último correo que te enviamos.

Si no es ninguna de las anteriores, puedes esperar 5 minutos y solicitar un nuevo Token usando la opción de recordar contraseña.
